# FPGA Egg Timer: Flexible Cooking Timer

## Overview
An **FPGA-based stopwatch** developed using **Verilog**, capable of showing time in **MM:SS:FF** format with the ability to **count down**. This project enables users to precisely control the timing of cooking with adjustable time features.

## Features
- **Flexible Time Control**: Users can add or subtract time, allowing for precise cooking control.
- **Finite State Machines (FSMs)**: Utilized for handling clock, button presses, and timing functionalities.
- **User-Friendly**: Provides buttons for setting, resetting, and changing time.

## Key Technologies
- **FPGA Development**
- **Verilog Programming Language**
- **Finite State Machines (FSM)**

## Functionality
The FPGA Egg Timer provides a flexible, user-controlled cooking timer that displays time in MM:SS:FF format. The timer can be adjusted in real-time, providing users with the ability to cook food to their desired level of doneness.
